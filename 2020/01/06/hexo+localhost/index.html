hexo+localhost

  安装git
https://git-scm.com/download/win
$ git –version
git version 2.21.0.windows.1

  安装node.js
https://nodejs.org/en/
$ node -v
v12.14.0
$ npm -v
6.13.4

  安装hexo
  可以更改为国内的镜像源
$ npm get registry 获得原来的镜像地址
https://registry.npmjs.org/
$ npm config set registry http://registry.npm.taobao.org/ 淘宝镜像
$ npm install -g hexo
$ hexo –version

  初始化
$ mkdir selfblog
$ cd selfblog/
$ hexo init

$ hexo s -g
